Check if your Keycloak token has the correct audience for the backend API

Instructions

  1. Make sure you're logged into the admin interface at http://localhost:5174
  2. Click the "Check Token" button below
  3. Review the token information to see if it includes the backend audience

What to Look For

Your token should include:

  • Audience (aud): Should contain both aws-framework-admin AND aws-framework-backend
  • Issuer (iss): Should be http://localhost:8080/realms/aws-framework
  • Roles: Should include admin or super-admin
üí° If the backend audience is missing:

You need to configure the Keycloak audience mapper. Follow the instructions in SUPER_ADMIN_FIX_INSTRUCTIONS.md
