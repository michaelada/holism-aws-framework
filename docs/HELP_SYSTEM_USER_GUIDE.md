# ItsPlainSailing Help System - User Guide

## Quick Start

ItsPlainSailing includes a built-in help system to guide you through using the platform. This guide explains how to use all the help features available to you.

---

## Welcome Dialog

### What It Is

A welcome dialog appears when you log in for the first time, introducing you to ItsPlainSailing and its key features.

### What You'll See

- Overview of the platform's capabilities
- Introduction to core modules (Users, Forms, Events, etc.)
- Information about getting started

### How to Dismiss

1. Read through the welcome information
2. **Optional**: Check "Don't show this again" if you don't want to see it on future logins
3. Click the close button or click outside the dialog

### Important Notes

- If you don't check "Don't show this again", the dialog will appear on your next login
- Your preference is saved to your account and applies across all devices
- You can always access help content through the help button (see below)

---

## Module Introduction Dialogs

### What They Are

Each module (Dashboard, Users, Forms, Events, Memberships, Calendar, Payments) has its own introduction dialog that appears the first time you visit it.

### What You'll Learn

- What the module does
- Key features available in that module
- How to get started with common tasks

### How to Dismiss

1. Read the module introduction
2. Click "Got it" or the close button
3. The introduction won't appear again for that module

### Important Notes

- Each module has a separate introduction
- Once dismissed, you won't see it again (even on other devices)
- Dismissing one module's introduction doesn't affect others

## Help Button and Drawer

### Finding the Help Button

Look for the **?** (question mark) button in the top right corner of the navigation bar, next to your user profile.

### Opening the Help Drawer

**Method 1**: Click the **?** button

**Method 2**: Press **Shift + ?** on your keyboard

### What's Inside

The help drawer provides:

- **Page-Specific Help**: Guidance for the exact page you're viewing
- **Module Help**: General information about the current module
- **Rich Content**: Well-formatted text with headings, lists, and helpful links

### Using the Help Drawer

**Reading Content**:
- Scroll through the help content to find what you need
- Content is organized with clear headings and sections

**Staying Productive**:
- The help drawer slides in from the right side
- You can still see and interact with the main content area
- No need to close help to continue working

**Closing the Drawer**:
- Click the close button (X) in the drawer header
- Click outside the drawer
- Press **Shift + ?** again
- Press **Esc** on your keyboard

**Reopening**:
- Click the help button anytime you need assistance
- Help content automatically updates based on your current page

## Managing Your Onboarding Preferences

### What Gets Saved

Your account stores:
- Whether you've dismissed the welcome dialog
- Which module introductions you've seen

### Where It Applies

- Your preferences sync across all devices
- Log in from anywhere and your preferences follow you

### Privacy

- Your preferences are stored securely with your account
- Only you can see your preferences
- Never shared with other users

### Resetting Preferences

Currently, you cannot reset onboarding preferences yourself. If you need to see the welcome dialog or module introductions again, contact your system administrator.

## Troubleshooting

### Help Button Not Visible

**Problem**: Can't find the help button

**Solution**: Look in the top right corner of the navigation bar, next to your profile. It's a **?** icon.

### Help Drawer Won't Open

**Problem**: Clicking help button does nothing

**Solutions**:
1. Refresh the page
2. Try the keyboard shortcut (**Shift + ?**)
3. Check if another dialog is open (close it first)
4. Clear browser cache
5. Try a different browser

### Wrong Help Content Showing

**Problem**: Help content doesn't match the current page

**Solutions**:
1. Close and reopen the help drawer
2. Refresh the page
3. Navigate away and back to the page
4. Report the issue to support

### Help Content in Wrong Language

**Problem**: Help appears in the wrong language

**Solutions**:
1. Check your language preference in profile settings
2. Change language and verify help updates
3. Some content may fall back to English if translations are incomplete

### Welcome Dialog Keeps Appearing

**Problem**: Welcome dialog shows every time I log in

**Solution**: Make sure to check "Don't show this again" before closing the dialog. The checkbox must be checked for your preference to be saved.

### Module Introduction Reappearing

**Problem**: Module introduction shows again after dismissing

**Solution**: This shouldn't happen. If it does:
1. Clear browser cache
2. Log out and log back in
3. Contact support if issue persists

## Frequently Asked Questions

**Q: Can I turn off all onboarding dialogs?**

A: Yes, check "Don't show this again" on the welcome dialog, and dismiss each module introduction as you encounter them. After that, you won't see any more onboarding dialogs.

**Q: Will I lose access to help content if I dismiss the dialogs?**

A: No! The help button is always available. Dismissing onboarding dialogs only stops them from appearing automatically - you can always access help content through the help drawer.

**Q: Can I see the welcome dialog again after dismissing it?**

A: Not directly. Contact your system administrator if you need to reset your onboarding preferences.

**Q: Does the help drawer work offline?**

A: No, you need an internet connection to use ItsPlainSailing and access help content.

**Q: Can I print help content?**

A: Yes, you can copy content from the help drawer and paste it into a document, or print this user guide for reference.

**Q: Is help content available in my language?**

A: Help content is available in English (en-GB), French (fr-FR), Spanish (es-ES), Italian (it-IT), German (de-DE), and Portuguese (pt-PT). Set your language preference in your profile settings.

**Q: How often is help content updated?**

A: Help content is updated whenever new features are added or existing features change. You'll always see the most current help content.

**Q: Can I suggest improvements to help content?**

A: Absolutely! We welcome feedback. Contact support with your suggestions.

The ItsPlainSailing help system provides three levels of assistance:

1. **Welcome Dialog**: One-time introduction for new users

Use the **?** button or press **Shift + ?** anytime you need help. The help system is designed to support your learning without interrupting your work.
