## Automatic Status Management

**Nightly Job (Midnight):**
- Runs automatically every night at 00:00
- Queries all members with status = 'active'
- Checks if validUntil < current date
- Updates status to 'elapsed' for expired members
- Logs all status changes for audit trail
- Sends optional notification to admins (configurable)

**Implementation Notes:**
- Use cron job or scheduled task
- Transaction-based updates for data integrity
- Error handling and retry logic
- Performance optimization for large member databases
